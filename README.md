# vue3-devbox

# 0. install devbox
`$ curl -fsSL https://get.jetpack.io/devbox | bash`

 Windowsの方はこちらを参照ください：https://zenn.dev/desse_nam2o/articles/e0832e5a1bfe44
 
# 1. open devbox shell
`$ devbox shell`


# 2. install modules
`$ cd app`

`$ npm install`

# 3. build
`$ npm run dev`
